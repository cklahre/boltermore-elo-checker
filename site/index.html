What’s here

Static views for leaderboard data built in CI from this repo. These pages do not include BCP roster lookup (elo-roster)—that still needs the CLI and a Best Coast Pairings event id.

  • Local match Elo — rankings from your exported matches JSON (same rules as local-elo).
  • Stat-Check Elo — snapshot of the global Stat-Check feed exported at deploy time (same source as statcheck-elo-leaderboard).
Deployed with GitHub Actions — data files live under data/.
